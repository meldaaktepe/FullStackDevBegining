Gloabal Atributes:
Atributes that can be applied to any HTML element.
class Atribute
Allows us to target all elements with that class in CSS or JavaScript 
    
      <p class="intro">
        This is the introduction
      </p>
id Atributes
Allows us to target a unique element with that id in our CSS of JavaScript
      
        <p class="intro" id="article-intro">
          This is the introduction
        </p>
id can be use to targeting CSS, but it is much more specific and that can be a problem. So most of the time developers tend to stick to classes. id's can be helpfull for addressing particular elements with JavaScript or with a targeted link. The fact that there will only ever be one element with any particular id name, becomes useful when interacting with JavaScript or links.
